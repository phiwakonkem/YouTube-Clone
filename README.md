# YouTube Clone

A static HTML/CSS recreation of the YouTube homepage — top navbar with search bar, collapsible sidebar with category icons, and a video grid layout — built as a front-end layout practice project.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)

> 📚 Learning project — not affiliated with YouTube/Google LLC. Logo/branding used for non-commercial, educational recreation only.

## Features

- Top nav: menu toggle, YouTube logo, search bar with voice-search icon, notifications badge, create button, and channel avatar
- Sidebar: Home / Shorts / Subscriptions, and a "You" section, using inline SVG icons matching YouTube's icon set
- Material Icons integration for search/notification/menu icons

## Tech Stack

HTML5 · CSS3 · Material Icons

No JavaScript or build tools — pure static HTML/CSS. Icons and the profile avatar are pulled from external image URLs rather than bundled locally.

## Getting Started

```bash
git clone https://github.com/phiwakonkem/YouTube-Clone.git
cd YouTube-Clone
```

Open `index.html` directly in your browser, or use the VS Code **Live Server** extension for auto-reload.
